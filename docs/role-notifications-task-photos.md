# Ролевые уведомления и контроль фото задач

- Лента колокольчика и push используют один ролевой фильтр.
- Руководитель получает все направления по умолчанию и может отключать отдельные роли.
- Прораб получает уведомления только по доступным объектам.
- Добавлены ежедневные операционные напоминания для прораба, HR, бухгалтера, юриста и руководителя.
- Новая задача проходит через скрытый черновик и публикуется только после загрузки фото «До».
- Статус «Выполнено» разрешён только после загрузки фото «После».
